# Calendar Domain

Skills for interacting with Google Calendar. Includes listing events within
a date range, creating new events with attendees, and updating existing events.

## Skill Inventory

| Skill | Type | Purpose |
|-------|------|---------|
| calendar.list | Read | List upcoming events with date/time filtering |
| calendar.create | Write | Create a new calendar event |
| calendar.update | Write | Update an existing calendar event |
