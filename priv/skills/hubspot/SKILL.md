# HubSpot Domain

Skills for interacting with HubSpot CRM. Includes creating, reading, updating,
deleting, searching, and listing recent records for contacts, companies, and deals.

## Skill Inventory

| Skill | Type | Purpose |
|-------|------|---------|
| hubspot.create_contact | Write | Create a new contact |
| hubspot.get_contact | Read | Get a contact by ID |
| hubspot.update_contact | Write | Update an existing contact |
| hubspot.delete_contact | Write | Archive (delete) a contact |
| hubspot.search_contacts | Read | Search contacts by email or name |
| hubspot.list_recent_contacts | Read | List recently created/updated contacts |
| hubspot.create_company | Write | Create a new company |
| hubspot.get_company | Read | Get a company by ID |
| hubspot.update_company | Write | Update an existing company |
| hubspot.delete_company | Write | Archive (delete) a company |
| hubspot.search_companies | Read | Search companies by name or domain |
| hubspot.list_recent_companies | Read | List recently created/updated companies |
| hubspot.create_deal | Write | Create a new deal |
| hubspot.get_deal | Read | Get a deal by ID |
| hubspot.update_deal | Write | Update an existing deal |
| hubspot.delete_deal | Write | Archive (delete) a deal |
| hubspot.search_deals | Read | Search deals by name or stage |
| hubspot.list_recent_deals | Read | List recently created/updated deals |
