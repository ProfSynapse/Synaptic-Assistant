# Email Domain

Skills for interacting with Gmail. Includes listing recent messages, searching
by query and filters, reading full message content by ID, creating drafts, and
sending new emails.

## Skill Inventory

| Skill | Type | Purpose |
|-------|------|---------|
| email.list | Read | List recent emails from a Gmail label |
| email.search | Read | Search Gmail messages by query, sender, date range |
| email.read | Read | Read a full email message by ID |
| email.draft | Read | Create a Gmail draft (saved, not sent) |
| email.send | Write | Send an email (requires confirmation) |
